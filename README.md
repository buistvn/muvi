## Getting Started

To get a local copy up and running, follow these instructions.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/buistvn/muvi.git
   ```
2. Install packages
   ```sh
   npm install
   ```



<!-- USAGE -->
## Usage

* Run the app in [development mode](http://localhost:3000/)
  ```sh
  npm start
  ```
* Generate a production build and deploy it to [GitHub Pages](https://buistvn.github.io/muvi/#/)
  ```sh
  npm run deploy
  ```
